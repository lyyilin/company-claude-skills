## 黄金3秒开场库

### 冲突开场型
- "离婚！马上给我签字！"（文件摔在桌上）
- "少爷，您...您竟然被退婚了！"（跪地震惊）

### 危机开场型
- "三、二、一...时间到，准备收尸。"（炸弹倒计时归零）
- "整个京圈都在找她，你竟然把她藏在这里？"（破门而入）

- "恭迎魔帝回归！"（万人跪拜）
- "这张黑卡...全球只发行了三张..."（服务员颤抖）

### 悬念开场型
- "DNA检测报告显示...他是你的亲生父亲。"
- "十年前你死的那天，我就在现场。"

## 自动修复策略

### 问题1：开场拖沓
**诊断**：前3秒无爆点
**修复**：
- 删除所有环境描写/铺垫
- 直接从冲突点开场
- 开场第一句话/画面必须有冲击力

### 问题2：爽点稀疏
**诊断**：30秒无反转/打脸
**修复**：
- 识别平淡段落
- 插入打脸桥段（实力碾压/身份揭露/证据亮出）
- 或插入意外反转（盟友背叛/隐藏身份暴露）

### 问题3：钩子缺失
**诊断**：结尾无S级悬念
**修复**：
- 强制使用S级钩子模板
- 或让主角做出意外选择（"既然如此，那就全灭吧"）

### 问题4：节奏拖沓
**诊断**：有无效画面/对话
**修复**：
- 删除所有非必要场景
- 每句话必须有推动力
- 删减冗余对白，保留核心冲突

### 问题5：情绪平淡
**诊断**：无法引起共鸣
**修复**：
- 强化极端情绪（极致愤怒/极致悲伤/极致惊喜）
- 用画面+音效双重刺激
- 打脸时刻必须有慢动作+重音效

## 质量熔断机制

- **IF 总分 ≥ 90分**：✅ 判定合格，输出剧本
- **IF 总分 < 90分**：❌ 判定不合格，触发自动修正
- **最大修正轮次**：3轮（防止死循环）

## 创作策略建议

2. **紧跟榜单** - 盯紧起点/番茄热门榜单
3. **数据驱动** - 小规模测试快速迭代

### 制作策略三要素
1. **小团队快迭代** - 5-8人团队效率最高
2. **AI全流程应用** - 从剧本到视频全面AI化
3. **多风格并行测试** - 同时测试多种风格选最优
