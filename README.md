# Company Claude Skills Library 🚀

欢迎来到公司内部的 Claude Skills 共享仓库！这里是团队成员分享和发现高质量 Claude 技能的中心。

## 📚 关于本仓库

本仓库用于存储和共享公司内部开发的 Claude Skills，帮助团队成员：
- 🔍 发现和使用他人创建的优秀技能
- 📤 分享自己开发的实用技能
- 🤝 协作改进现有技能
- 📖 学习最佳实践和开发模式

## 🎯 快速开始

### 使用现有 Skill

1. 浏览 [`skills/`](./skills/) 目录查看所有可用技能
2. 阅读技能的 `SKILL.md` 了解功能和用法
3. 将技能目录复制到你的项目的 `.claude/skills/` 目录下
4. 开始使用！

### 上传你的 Skill

详细步骤请查看 [上传指南](./docs/how-to-upload.md)

**快速步骤：**
1. 使用 [`skills/_template/`](./skills/_template/) 作为模板
2. 按照 [Skill 开发规范](./docs/skill-guidelines.md) 创建你的技能
3. Fork 本仓库并创建新分支
4. 将你的技能添加到 `skills/` 目录
5. 提交 Pull Request

## 📂 仓库结构

```
company-claude-skills/
├── skills/              # 所有技能存放目录
│   ├── _template/      # 技能开发模板
│   └── example-skill/  # 示例技能
├── docs/               # 文档
│   ├── how-to-upload.md      # 上传指南
│   └── skill-guidelines.md   # 开发规范
└── scripts/            # 辅助脚本
```
